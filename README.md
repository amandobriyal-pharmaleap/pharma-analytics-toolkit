# Pharma Analytics Toolkit

A practical starter toolkit for pharma analytics workflows, including Jupyter notebooks for KPI tracking and exploratory analysis.

## What's Inside

- notebooks/01_welcome.ipynb: quick orientation and workflow setup
- notebooks/02_kpi_starter.ipynb: starter pharma KPI calculations
- data/sample_sales.csv: small sample dataset for experimentation
